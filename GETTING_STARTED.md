## Quick Start

```bash
# Install dependencies
npm install

# Start Docker services (PostgreSQL + MinIO)
docker compose up -d

# Setup database
npm run db:generate
npm run db:push
npm run db:seed

# Run development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)
